# Why Clay is a POWERFUL idea

The reason Clay is clicking for a lot of people is because it represents flows in a very intuitive way: a google sheet.

Each row of this google sheet is a an object that went through a flow. And you can see all the inputs and outputs of each node of this state machine.

So in a way, Clay is a log table.

The next step for clay is to bring [human in the loop](/blog/lpws-with-humans-in-the-loop) and go [compound](/blog/compound-startups) to provide a full customer experience, and not just help with outbound and enrichments.
