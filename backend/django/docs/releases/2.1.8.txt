==========================
Django 2.1.8 release notes
==========================

*April 1, 2019*

Django 2.1.8 fixes a bug in 2.1.7.

Bugfixes
========

* Prevented admin inlines for a ``ManyToManyField``\'s implicit through model
  from being editable if the user only has the view permission
  (:ticket:`30289`).
